# Changelog

## 0.2.4 — 2026-07-02

- Добавлена поддержка тапа по карточкам в разделе «Офлайн» на iPhone.
- Тап устанавливает фокус и вызывает то же действие, что `OK / Enter`.
- Добавлена защита от двойного события `touchend + click`.
- Свайп для прокрутки списка не считается нажатием.
- Стабильный загрузчик переключён на версию 0.2.4.

## 0.2.3 — 2026-07-02

- Обычный `OK / Enter` теперь открывает меню действий для любой записи.
- Для статусов `queued` и `downloading` добавлено действие «Отменить и удалить».
- Активный сетевой запрос прерывается через `AbortController`.
- Удаляются недогруженные HLS-фрагменты, файл и метаданные из IndexedDB.
- Запись можно удалить из очереди до начала загрузки.
- Долгое нажатие открывает то же меню, но больше не является обязательным.

## 0.2.2 — 2026-07-02

- Сохранённый `blob:...#offline.m3u8` принудительно открывается через Hls.js.
- Исправлена ошибка `The operation is not supported` при локальном HLS-воспроизведении.
- Широкий глобальный CORS-перехват заменён на проверку вызовов только из ядра офлайн-загрузчика.
- Убраны сотни повторяющихся сообщений о переключении credentials.

## 0.2.1 — 2026-07-02

- Исправлена ошибка `Lampa.Controller.active is not a function` в разделе «Офлайн».
- Для медиазапросов загрузчика теперь используется `credentials: omit`.
- Исправлена загрузка MP4 с серверов, которые возвращают `Access-Control-Allow-Origin: *`.
- Добавлена совместимость с текущим API контроллера Lampa.

## 0.2.0 — 2026-07-02

- Добавлен перехват HLS-потоков `.m3u8`.
- Добавлена загрузка master/media-плейлистов и HLS-фрагментов.
- Выбирается вариант с наибольшим `BANDWIDTH`.
- Добавлена поддержка незашифрованного `EXT-X-MAP`.
- Добавлено локальное воспроизведение сохранённого HLS.
- Зашифрованные HLS, DRM и `EXT-X-BYTERANGE` явно отклоняются.
- Создан отдельный установочный файл, чтобы исключить кэш версии 0.1.0.

## 0.1.0 — 2026-07-02

- Перехват запуска прямого видео через `Lampa.Player.listener`.
- Выбор: смотреть, скачать, смотреть и скачать.
- Последовательная очередь загрузок.
- Хранение в OPFS с fallback на IndexedDB.
- Раздел «Офлайн» в главном меню.
- Запуск, отмена, повтор и удаление файлов.
- Настройки запроса перед воспроизведением и очистки хранилища.
