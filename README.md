# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.3.1

Версия 0.3.1 исправляет источники, которые сначала возвращают JSON-манифест с перечнем качеств, а не готовое видео.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите предыдущую версию офлайн-плагина.
3. Добавьте:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.3.1.js
```

4. Полностью закройте Lampa и откройте снова при включённом интернете.

## Меню перед запуском видео

Плагин добавляет действия:

- **Смотреть**;
- **Сохранить офлайн**;
- **Скачать на устройство**;
- **Смотреть и сохранить**.

## JSON-манифесты

Некоторые источники передают в плеер не видео, а служебный API-адрес. Ответ содержит JSON примерно такого вида:

```json
{
  "sources": [
    {
      "links": [
        {"quality": "720", "src": "https://example/720/master.m3u8"},
        {"quality": "1080", "src": "https://example/1080/master.m3u8"}
      ]
    }
  ]
}
```

Версия 0.3.1:

- не открывает такой JSON как скачанный файл;
- извлекает `src`, `url`, `link` и `file` из ответа;
- показывает выбор качества;
- для прямого MP4/WebM/MKV предлагает открыть файл на устройстве;
- для `.m3u8` предлагает **Сохранить офлайн** или **Смотреть**.

## Скачать на устройство

Для прямого MP4/WebM/MKV:

1. выберите **Скачать на устройство**;
2. плагин определит реальную ссылку;
3. нажмите **Открыть видео для сохранения**;
4. в системном просмотрщике iPhone выберите **Поделиться → Сохранить в Файлы**.

Файл не добавляется в IndexedDB.

## HLS

Ссылку `.m3u8` нельзя сохранить как один готовый файл простым открытием. Это плейлист из множества фрагментов.

Для HLS версия 0.3.1 показывает качество и затем предлагает:

- **Сохранить офлайн** — поток будет загружен внутрь Lampa;
- **Смотреть** — открыть выбранное качество.

После завершения загрузки HLS можно экспортировать через:

```text
Офлайн → В Файлы
```

Экспорт HLS является best-effort: отдельные потоки могут не открыться вне Lampa после простого объединения фрагментов.

## Кнопка «В Файлы»

Для уже сохранённых записей доступны:

- **Поделиться / Сохранить в Файлы** — если устройство поддерживает передачу файла;
- **Открыть файл**;
- **Скачать**.

## Минимальный офлайн-интерфейс

Когда Lampa уже находится в памяти и интернет пропадает, базовая версия 0.2.6 показывает собственный экран **Офлайн** и читает загрузки из `lampa-offline-hls-v2`.

Плагин не может гарантировать холодный запуск после полного закрытия приложения, если сам HTML сайта Lampa не загрузился без сети. Для этого нужен service worker на домене Lampa.

## Ограничения форматов

Не поддерживаются:

- DRM/Widevine;
- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- MPEG-DASH (`.mpd`);
- отдельная внешняя аудиодорожка из сложного master-плейлиста;
- источники, требующие специальных HTTP-заголовков.

## Диагностика

После запуска:

```text
[Offline Device] player source recorder installed
[Offline Device] initialized 0.3.1
```

После разбора служебного ответа:

```text
[Offline Device] source resolved <количество>
```

При автоматической передаче выбранного HLS обратно загрузчику:

```text
[Offline Device] automatic resolved action save <url>
```
