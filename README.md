# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.2.6

Версия 0.2.6 добавляет минимальный офлайн-интерфейс. Если Lampa уже смогла запуститься без сети, плагин не ждёт загрузки каталога и сразу показывает сохранённые видео из IndexedDB.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите предыдущую версию офлайн-плагина.
3. Добавьте:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.2.6.js
```

4. Полностью закройте Lampa и откройте снова.
5. Пока интернет доступен, один раз откройте раздел **Офлайн**, чтобы убедиться, что плагин запустился и файлы видны.

## Что происходит без интернета

Когда `navigator.onLine === false`, версия 0.2.6:

- не загружает основное ядро офлайн-плагина с CDN;
- перекрывает обычный интерфейс Lampa собственным полноэкранным экраном **Офлайн**;
- читает список напрямую из базы `lampa-offline-hls-v2`;
- показывает готовые, недогруженные и ошибочные записи;
- позволяет смотреть готовое видео и удалять любую запись;
- использует собственный `<video>`-плеер, а не навигацию Lampa;
- автоматически возвращает обычный интерфейс после восстановления сети.

Для MP4/WebM/MKV создаётся локальный `blob:` URL. Для HLS плагин собирает локальный плейлист из сохранённых фрагментов и использует Hls.js, если он доступен.

## Ограничение холодного запуска

Версия 0.2.6 решает сценарий, когда установленное веб-приложение или браузер уже смогли восстановить HTML и выполнить сохранённый плагин.

Она не может гарантировать запуск после полного выгружения приложения, если сам сайт Lampa не открылся без сети. Для этого нужен service worker на домене Lampa. Внешний плагин не может установить service worker для чужого origin.

Плагин дополнительно сохраняет собственный файл и онлайн-часть в Cache API как best-effort кэш, но без service worker браузер не обязан использовать этот кэш для холодного запуска страницы.

## Онлайн-режим

При наличии сети версия 0.2.6 загружает версию 0.2.5 и сохраняет все функции:

- перехват прямых MP4/WebM/MKV;
- загрузка незашифрованного HLS;
- очередь и прогресс;
- повтор, отмена и удаление;
- сенсорное управление на iPhone;
- прямой запуск сохранённого видео из IndexedDB.

## Ограничения форматов

Не поддерживаются:

- DRM/Widevine;
- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- MPEG-DASH (`.mpd`);
- отдельная внешняя аудиодорожка из сложного master-плейлиста;
- источники, требующие специальных HTTP-заголовков.

Ответ `423 Locked` приходит от видеосервера. Такую ссылку нельзя скачать до получения новой рабочей ссылки от онлайн-плагина.

## Диагностика

При обычном онлайн-запуске:

```text
[Offline Shell] initialized 0.2.6 online: true
[Offline Shell] online plugin initialized 0.2.6
```

При переходе в офлайн:

```text
[Offline Shell] minimal offline shell shown 0.2.6
```

При возвращении сети:

```text
[Offline Shell] minimal offline shell hidden
```
