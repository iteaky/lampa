# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.3.2

Версия 0.3.2 добавляет прямое сохранение в загрузки устройства без промежуточного помещения файла в IndexedDB.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите предыдущую версию офлайн-плагина.
3. Добавьте:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.3.2.js
```

4. Полностью закройте Lampa и откройте снова при включённом интернете.

## Меню перед запуском видео

Плагин добавляет действия:

- **Смотреть**;
- **Сохранить офлайн**;
- **Скачать в Загрузки**;
- **Смотреть и сохранить**.

## Скачать в Загрузки

После выбора **Скачать в Загрузки** плагин:

1. определяет реальный источник и разбирает служебный JSON;
2. показывает доступные качества;
3. загружает прямой видеофайл или все HLS-фрагменты во временную память;
4. собирает `.mp4`, `.ts`, `.webm`, `.mkv` или `.mov`;
5. показывает явную кнопку **Сохранить в Загрузки**.

Файл не добавляется в раздел **Офлайн** и не сохраняется в IndexedDB.

На iPhone после нажатия **Сохранить в Загрузки** Safari может показать системное подтверждение загрузки. Без пользовательского нажатия веб-приложение не может молча записать файл в папку устройства.

## JSON-манифесты

Если источник сначала возвращает JSON со ссылками `src`, `url`, `link` или `file`, плагин извлекает реальные варианты и показывает выбор качества.

## HLS

Для незашифрованного HLS версия 0.3.2:

- загружает master/media-плейлист;
- выбирает вариант с наибольшим `BANDWIDTH` внутри выбранного качества;
- загружает init-сегмент и все видеофрагменты;
- собирает fragmented MP4 в `.mp4`;
- собирает MPEG-TS в `.ts`;
- показывает прогресс и кнопку отмены.

Не поддерживаются:

- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- HLS с отдельной внешней аудиодорожкой;
- DRM/Widevine;
- MPEG-DASH (`.mpd`);
- источники, требующие специальных HTTP-заголовков.

Сборка HLS является best-effort: отдельные потоки могут не открыться во внешнем плеере после простого объединения фрагментов.

## Ограничения iPhone

Во время подготовки весь файл находится в памяти веб-приложения. Для длинного фильма или высокого качества iOS может закрыть вкладку из-за нехватки памяти. Не закрывайте Lampa до появления кнопки **Сохранить в Загрузки**.

Если сервер запрещает CORS-запросы, плагин не сможет скачать файл через JavaScript.

## Кнопка «В Файлы»

Для уже сохранённых внутри Lampa записей остаются действия:

- **Поделиться / Сохранить в Файлы**;
- **Открыть файл**;
- **Скачать**.

## Минимальный офлайн-интерфейс

Когда Lampa уже находится в памяти и интернет пропадает, базовая версия 0.2.6 показывает собственный экран **Офлайн** и читает загрузки из `lampa-offline-hls-v2`.

Плагин не может гарантировать холодный запуск после полного закрытия приложения, если сам HTML сайта Lampa не загрузился без сети. Для этого нужен service worker на домене Lampa.

## Диагностика

После запуска:

```text
[Offline Device] player source recorder installed
[Offline Device] initialized 0.3.2
```

После подготовки файла:

```text
[Offline Device] device file prepared <имя> <размер>
```

При ошибке:

```text
[Offline Device] device file failed <сообщение>
```
