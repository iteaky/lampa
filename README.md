# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.2.4

Версия 0.2.4 исправляет управление списком **Офлайн** на iPhone и других сенсорных устройствах.

Если установлена предыдущая версия, сначала удалите её из списка расширений Lampa. Одновременно оставлять несколько версий нельзя.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите старый адрес офлайн-плагина.
3. Нажмите **Добавить плагин**.
4. Вставьте адрес:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.2.4.js
```

5. Подтвердите добавление.
6. Полностью закройте Lampa, выгрузите её из памяти и откройте снова.

## Как пользоваться

1. Откройте фильм или сериал через онлайн-плагин.
2. Выберите перевод, качество или серию и запустите видео.
3. Перед открытием плеера появится меню:
   - **Смотреть**;
   - **Сохранить офлайн**;
   - **Смотреть и сохранить**.
4. Прогресс отображается в разделе **Офлайн**.

## Действия с загрузкой

На телевизоре или компьютере выделите запись и нажмите **OK / Enter**.

На iPhone просто коснитесь карточки загрузки. Тап теперь:

- устанавливает фокус на выбранную запись;
- вызывает то же действие, что кнопка **OK**;
- не срабатывает после прокрутки списка;
- защищён от двойного события `touchend + click`.

Для готовой записи доступны:

- **Смотреть офлайн**;
- **Удалить**.

Для загрузки со статусом **В очереди** или **Загрузка** доступны:

- **Отменить и удалить**;
- **Удалить**.

Для записи со статусом **Ошибка** или **Отменено** доступны:

- **Повторить загрузку**;
- **Удалить**.

## Что исправлено в 0.2.4

- добавлена обработка `touchstart`, `touchmove`, `touchend` и обычного `click`;
- тап по карточке на iPhone открывает меню действий;
- свайп для прокрутки списка не считается нажатием;
- предотвращено двойное открытие меню после `touchend` и последующего `click`;
- сохранены исправления отмены загрузки, удаления недогруженных частей, локального HLS и CORS из версий 0.2.1–0.2.3;
- стабильный файл `lampa-offline-mvp.js` переключён на 0.2.4.

## Возможности версии 0.2.x

- прямые MP4, WebM и MKV;
- незашифрованные HLS-потоки `.m3u8`;
- выбор варианта с наибольшим `BANDWIDTH`;
- загрузка HLS-плейлиста и видеофрагментов;
- поддержка незашифрованного `EXT-X-MAP`;
- локальное воспроизведение через Hls.js;
- очередь, прогресс, повтор, отмена и удаление загрузок.

## Ограничения

Не поддерживаются:

- DRM/Widevine;
- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- MPEG-DASH (`.mpd`);
- отдельная внешняя аудиодорожка из сложного master-плейлиста;
- источники, требующие специальных HTTP-заголовков;
- загрузка при полностью закрытой Lampa;
- продолжение с точного байта после закрытия приложения.

Ответ `423 Locked` приходит от самого видеосервера. Такую ссылку нельзя скачать до получения новой рабочей ссылки от онлайн-плагина.

## Диагностика

После запуска версии 0.2.4 в консоли должна появиться строка:

```text
[Offline HLS touch] touch and click actions installed 0.2.4
```

После тапа по записи:

```text
[Offline HLS touch] offline item activated by touch
```
