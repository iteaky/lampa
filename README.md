# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.3.0

Версия 0.3.0 добавляет отдельный вариант сохранения прямого видео сразу на устройство, без помещения файла в раздел **Офлайн**.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите предыдущую версию офлайн-плагина.
3. Добавьте:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.3.0.js
```

4. Полностью закройте Lampa и откройте снова при включённом интернете.

## Новое меню перед запуском видео

Для прямой HTTP/HTTPS-ссылки отображаются действия:

- **Смотреть**;
- **Сохранить офлайн** — сохранить внутри Lampa;
- **Скачать на устройство** — открыть исходный файл в системном просмотрщике;
- **Смотреть и сохранить**.

После выбора **Скачать на устройство** прямой MP4/WebM/MKV открывается отдельно. На iPhone нажмите:

1. **Поделиться**;
2. **Сохранить в Файлы**.

Файл не добавляется в IndexedDB и не появляется в разделе **Офлайн**.

Если iPhone заблокирует автоматическое открытие, плагин покажет экран с кнопкой **Открыть видео для сохранения**.

## HLS

Ссылку `.m3u8` нельзя сохранить на устройство как один готовый видеофайл прямым открытием: она является плейлистом из множества фрагментов.

Для HLS используется прежний порядок:

1. **Сохранить офлайн**;
2. открыть раздел **Офлайн**;
3. нажать **В Файлы**;
4. подготовить `.mp4` или `.ts`;
5. сохранить через системный просмотрщик iPhone.

Экспорт HLS является best-effort: отдельные потоки могут не открыться вне Lampa после простого объединения фрагментов.

## Кнопка «В Файлы»

Для уже сохранённых записей версия 0.2.9, входящая в 0.3.0, сохраняет действия:

- **Поделиться / Сохранить в Файлы** — если устройство поддерживает передачу файла;
- **Открыть файл**;
- **Скачать**.

## Минимальный офлайн-интерфейс

Когда Lampa уже находится в памяти и интернет пропадает, базовая версия 0.2.6 показывает собственный экран **Офлайн** и читает загрузки из `lampa-offline-hls-v2`.

Плагин не может гарантировать холодный запуск после полного закрытия приложения, если сам HTML сайта Lampa не загрузился без сети. Для этого нужен service worker на домене Lampa.

## Ограничения форматов

Не поддерживаются:

- DRM/Widevine;
- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- MPEG-DASH (`.mpd`);
- отдельная внешняя аудиодорожка из сложного master-плейлиста;
- источники, требующие специальных HTTP-заголовков.

## Диагностика

После запуска версии 0.3.0:

```text
[Offline Device] player source recorder installed
[Offline Device] initialized 0.3.0
```

Перед показом меню:

```text
[Offline Device] device option added to player menu
```

При выборе прямого сохранения:

```text
[Offline Device] direct device option <url>
```
