# Lampa Offline MVP

Плагин добавляет в Lampa сохранение прямых видеофайлов и незашифрованных HLS-потоков для просмотра без интернета.

> Используйте плагин только для контента, который разрешено сохранять. Он не обходит DRM, шифрование и защиту видеосервисов.

## Установка версии 0.2.9

Версия 0.2.9 исправляет экспорт сохранённого видео на iPhone, когда кнопка открытия системного меню ничего не делала.

1. Откройте **Lampa → Настройки → Расширения**.
2. Удалите предыдущую версию офлайн-плагина.
3. Добавьте:

```text
https://cdn.jsdelivr.net/gh/iteaky/lampa@main/lampa-offline-mvp-0.2.9.js
```

4. Полностью закройте Lampa и откройте снова при включённом интернете.

## Кнопка «В Файлы»

На обычной и минимальной странице **Офлайн** появляется кнопка **В Файлы**.

После выбора видео и подготовки файла версия 0.2.9 показывает доступные действия:

- **Поделиться / Сохранить в Файлы** — только если браузер явно сообщает, что поддерживает передачу этого файла;
- **Открыть файл** — настоящая ссылка на локальный `blob:` URL;
- **Скачать** — обычная ссылка с атрибутом `download` для браузеров, где она поддерживается.

На iPhone наиболее надёжный порядок:

1. нажмите **Открыть файл**;
2. в системном просмотрщике нажмите кнопку **Поделиться**;
3. выберите **Сохранить в Файлы**.

Если системное меню отклоняет файл или файл слишком большой, плагин теперь показывает текст ошибки и предлагает открыть файл напрямую вместо молчаливого отказа.

Видео не лежат в обычной папке iPhone: внутри Lampa они хранятся в IndexedDB. Поэтому плагин экспортирует выбранное видео во внешний файл.

### Прямые файлы

MP4, WebM и MKV экспортируются без изменения содержимого. Имя файла формируется из названия видео.

### HLS

HLS хранится набором фрагментов. При экспорте плагин пытается собрать их в один файл:

- поток с `EXT-X-MAP` — в `.mp4`;
- MPEG-TS — в `.ts`.

Это best-effort экспорт: отдельные HLS-потоки могут не открыться во внешнем плеере после простого объединения фрагментов. Внутри Lampa сохранённый HLS продолжает воспроизводиться через локальный плейлист.

## Минимальный офлайн-интерфейс

Если Lampa уже смогла запуститься без сети, базовая версия 0.2.6:

- перекрывает обычный интерфейс собственным экраном **Офлайн**;
- читает список из базы `lampa-offline-hls-v2`;
- позволяет смотреть и удалять сохранённые записи;
- показывает кнопку **В Файлы**;
- автоматически возвращает обычный интерфейс после восстановления сети.

## Ограничение холодного запуска

Плагин не может гарантировать запуск после полного закрытия приложения, если сам HTML сайта Lampa не загрузился без сети. Для этого нужен service worker на домене Lampa.

## Ограничения форматов

Не поддерживаются:

- DRM/Widevine;
- HLS с `EXT-X-KEY` и шифрованием;
- HLS с `EXT-X-BYTERANGE`;
- MPEG-DASH (`.mpd`);
- отдельная внешняя аудиодорожка из сложного master-плейлиста;
- источники, требующие специальных HTTP-заголовков.

## Диагностика

После запуска версии 0.2.9:

```text
[Offline Export] initialized 0.2.9
[Offline Export] base initialized 0.2.9
```

После подготовки файла:

```text
[Offline Export] prepared <имя файла> <размер>
```

При ошибке системного меню:

```text
[Offline Export] share failed <тип ошибки> <сообщение>
```
